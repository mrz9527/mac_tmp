# 启动服务
1. 运行python inference_server.py启动服务
2. http接口输入语句
例如：
{'recordid':recordid, "sentence": "国允订全球最低企业税率15仍挨批让步过多 中央社"}
3. 利用post请求访问：http://54.212.154.80:8081/demo
4. 得到结果如下：
{'recordid': '3091', 'result': '敏感'}
